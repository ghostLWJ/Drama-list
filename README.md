# Drama-list

## Getting Started

### Environment

NodeJS version: 8.9.4

npm version: 5.6.0

Vue-cli version: 2.9.1

### Installation

```bash
git clone https://github.com/ghostLWJ/Drama-list.git
cd Drama-list
npm install
```

### Run

```bash
npm run dev
```

Open your browser connect to http://localhost:8080
